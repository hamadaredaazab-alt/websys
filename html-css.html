Introduksjon

HTML (HyperText Markup Language) og CSS (Cascading Style Sheets) utfyller hverandre, men har ulike roller. HTML gir struktur og innhold til en nettside – uten HTML ville det ikke vært noe tekst, bilder eller lenker. CSS derimot tar seg av presentasjonen og gjør at innholdet ser ryddig og estetisk ut. En nettside med kun HTML fungerer teknisk, men fremstår naken og lite brukervennlig. Med CSS kan man kontrollere farger, fonter, layout og responsivitet, noe som er avgjørende for moderne webdesign. Kort sagt: HTML bygger grunnmuren, mens CSS gjør bygget attraktivt og tilpasset brukeren.

Eksempel

<!DOCTYPE html>
<html>
<head>
<title>Min side</title>
</head>
<body>
<h1>Hei verden!</h1>
</body>
</html>

Her kan du legge til dine egne notater eller kodesnutter for oppgaven.
